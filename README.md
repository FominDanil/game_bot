# Game bot
Autoplaying bot for KarateKido Telegramm game

![](https://i.imgur.com/RGi4x9F.gif)

## Requirements
- Python 3.10

## Installation

First, you need to clone this repository:

```bash
git clone https://github.com/FominDanil/game_bot.git
cd game_bot
```
## Usage
```bash
pip install -r requirements.txt
python main.py
```
